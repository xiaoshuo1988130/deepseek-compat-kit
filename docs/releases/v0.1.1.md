# DeepSeek CompatKit v0.1.1

Hotfix release for diagnostic accuracy.

## Install

```bash
npx deepseek-compat-kit@0.1.1 --help
```

## Fixes

- Fixed strict schema lint accuracy.
- Stopped reporting DeepSeek-supported JSON Schema constraints as unsupported:
  - `pattern`
  - `format`
  - `minimum`
  - `maximum`
  - `multipleOf`
- Kept the unsupported strict schema checks focused on fields that DeepSeek does not support, including:
  - `minLength`
  - `maxLength`
  - `minItems`
  - `maxItems`
- Deduplicated shared `reasoning_content` when one assistant message contains multiple tool calls that map to the same cached reasoning state.

## Tests

- Added a regression test for supported strict schema constraints.
- Added a regression test for multiple tool calls sharing one reasoning payload.
- CI passed for this release.

## Notes

The proxy behavior is validated against the repository mock upstream. Live DeepSeek regression tests are still pending.

# Contributing

DeepSeek CompatKit is currently focused on a narrow first milestone:

- Diagnose `reasoning_content` round-trip failures.
- Lint DeepSeek strict-mode tool schemas.
- Generate sanitized replay fixtures.
- Provide a minimal local proxy for migration relief.

Useful early contributions:

- Add sanitized failure fixtures.
- Improve schema lint rules.
- Improve docs for known DeepSeek V4 migration errors.
- Add provider/framework compatibility notes.

Please do not include API keys, full prompts, private tool results, or full `reasoning_content` in public issues.
